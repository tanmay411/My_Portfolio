# Personal Portfolio - Tanmay Srivastava

A professional, responsive portfolio website designed to showcase my technical skills, projects, and professional background.

## 🚀 Live Demo
Check it out here: [tanmay-srivastava.kryotek.in](http://tanmay-srivastava.kryotek.in)

## 🛠️ Tech Stack
* **Frontend:** HTML5, CSS3, JavaScript
* **Framework:** Bootstrap 5 (Responsive Design)
* **Backend:** PHP (Mail handling and dynamic components)
* **Deployment:** Hosted via Kryotek Softwares

## ✨ Key Features
* **Dynamic Contact Form:** Integrated with PHP for direct communication.
* **Responsive UI:** Optimized for all devices using Bootstrap grids.
* **Project Showcase:** Highlighted work including the **21BCE** brand strategy and web development.
* **SEO Optimized:** Clean code structure for better search visibility.

## 📁 Repository Structure
* `/assets` - Images, CSS, and JS files.
* `index.php` - Main landing page.
* `contact.php` - Backend logic for form handling.
